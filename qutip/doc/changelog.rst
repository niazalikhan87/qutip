+++++++++++++++++++++++++++++++++++++++++++

**THIS RELEASE NOW REQUIRES THE GCC COMPILER TO BE INSTALLED**

New Functions
---------------

SVN-1054: tidyup function to remove small elements from a Qobj.

SVN-1051: Added concurrence function.
