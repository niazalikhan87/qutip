QuTiP classes¶

The Qobj class¶

class qutip.Qobj(inpt=array([[0]]), dims=[[][]], shape=[])¶
@brief A constructor class for the quantum object (Qobj) base object.
Defines quantum operators and states. Includes class dependent math operations.
conj()¶

@brief Returns the conjugate operator of a given quantum object.

dag()¶

@brief Returns the adjont operator (dagger) of a given quantum object.

diag()¶

@brief Returns diagonal elements of object

eigenstates()¶

@brief Find the eigenstates and eigenenergies (defined for operators and superoperators)

expm()¶

@brief Returns a quantum object corresponding to the matrix exponential of a given square operator.

full()¶

Returns a dense array from quantum object data

norm()¶

@brief Returns norm of a quantum object. Norm is L2-norm for kets and trace-norm for operators.

sqrtm()¶

@brief Returns the operator corresponding to the sqrt of a given square operator.

tr()¶

@brief Returns the trace of a quantum object

unit()¶

@brief Returns the operator normalized to unity.

class qutip.eseries(q=array([], dtype=float64), s=array([], dtype=float64))¶

@brief Class representation of an exponential-series expansion of time-dependent quantum objects.

spec(es, wlist)¶

Evaluate the spectrum of an exponential series at frequencies in wlist.

tidy(*args)¶

return a tidier version of self

value(tlist)¶

Evaluate an exponential series at the times listed in tlist.
